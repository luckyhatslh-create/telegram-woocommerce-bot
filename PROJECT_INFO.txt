🚀 БЫСТРЫЙ СТАРТ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Установка зависимостей:
   bash install.sh

2. Настройка .env файла:
   cp .env.example .env
   nano .env  # Добавьте свои API ключи

3. Проверка подключения:
   python check_api.py

4. Запуск бота:
   python bot.py

• TELEGRAM_CONFIG - настройки бота
• LIMITS - лимиты и ограничения
